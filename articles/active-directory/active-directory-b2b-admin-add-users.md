# <a name="how-do-azure-active-directory-admins-add-b2b-collaboration-users"></a>Como é que os administradores do Azure Active Directory adicionam utilizadores de colaboração B2B?

Global admins limitado e pode utilizar o [portal do Azure](https://portal.azure.com) para convidar utilizadores de colaboração do B2B para o diretório, a nenhum grupo ou para qualquer aplicação.

## <a name="admins-adding-guest-users-to-the-directory"></a>Adicionar utilizadores de convidado para o diretório de administradores
Adicione utilizadores de colaboração do B2B para o diretório como um administrador do Azure AD, conforme mostrado no vídeo seguinte:

>[!VIDEO https://channel9.msdn.com/Blogs/Azure/Admin-adding-guests-for-B2B-collaboration-to-Azure-AD-tenant/Player]


## <a name="admins-adding-guest-users-to-a-group"></a>Adicionar utilizadores convidados a um grupo de administradores
Adicione utilizadores de colaboração do B2B a um grupo como um administrador do Azure AD, conforme mostrado no vídeo seguinte:

>[!VIDEO https://channel9.msdn.com/Blogs/Azure/admin-adding-guests-to-a-group/Player]


## <a name="admins-adding-guest-users-to-an-application"></a>Administradores de adicionar utilizadores de convidado para uma aplicação

Adicione utilizadores de colaboração do B2B a uma aplicação como um administrador do Azure AD, conforme mostrado no vídeo seguinte:

>[!VIDEO https://channel9.msdn.com/Blogs/Azure/admin-assign-b2b-to-apps/Player]

## <a name="admins-resending-invitations-to-guest-users"></a>Administradores enviar convites para os utilizadores convidados
Pode aceder à página de perfil de um utilizador colaboração B2B e reenvie convites para os utilizadores convidados não ainda resgatadas:

![reenvie convite](./media/active-directory-b2b-admin-add-users/resend-invitation.png)

> ! [NOTA] Se reenviar convites, o convite é enviado ao utilizador com sessão iniciada para utilizadores individuais, mesmo que o convite original foi enviado para uma aplicação específica ou um grupo.
