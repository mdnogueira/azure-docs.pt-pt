# <a name="sync-service-manager-metaverse-search"></a>Pesquisa de Metaverso de Gestor do serviço de sincronização

![Gestor do serviço de sincronização](./media/active-directory-aadconnectsync-service-manager-ui/mvsearch.png)

O separador de pesquisa de metaverso é útil para resolver problemas relacionados com a dados. Na metade superior, pode criar uma consulta com base numa combinação de atributos. Quando estiver satisfeito com a sua consulta, clique em **pesquisa**. O resultado é visível na grelha na parte inferior. Pode selecionar as colunas que devem estar visíveis com **definições da coluna**.

Nos resultados da pesquisa, selecione um objeto e **propriedades** para ver o [as propriedades do objeto de metaverso](active-directory-aadconnectsync-troubleshoot-object-not-syncing.md#metaverse-object-properties).
