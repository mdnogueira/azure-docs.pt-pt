# <a name="saas-application-integration-with-azure-active-directory"></a>Integração de aplicações de SaaS no Azure Active Directory

Para ajudar a integrar a sua capacidade de nuvem [(SaaS)](https://azure.microsoft.com/overview/what-is-saas/) aplicações com o Azure Active Directory, desenvolvemos uma coleção de tutoriais guiá-lo através da configuração.

Em que temos orientações de aprovisionamento de utilizador, incluímos um tutorial de aplicação para o aprovisionamento de utilizador na coluna da direita, junto do tutorial de aplicação.

Para obter a lista completa de aplicações SaaS que tenham sido previamente integradas com o Azure AD, consulte o [Active Directory Marketplace](https://azuremarketplace.microsoft.com/marketplace/apps/category/azure-active-directory-apps).
