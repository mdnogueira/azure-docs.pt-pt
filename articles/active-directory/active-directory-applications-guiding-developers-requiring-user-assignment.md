# <a name="azure-ad-and-applications-require-user-assignment"></a>Azure AD e aplicações: requerem a atribuição de utilizadores
## <a name="requiring-user-assignment"></a>Exigir a atribuição de utilizadores
1. Inicie sessão no portal do Azure com uma conta de administrador.
2. Clique em de **todos os itens** item de menu principal.
3. Escolha o diretório que está a utilizar para a aplicação.
4. Clique em de **aplicações** separador.
5. Selecione a aplicação na lista de aplicações associadas a este diretório.
6. Clique em de **configurar** separador.
7. Alterar o **utilizador atribuição necessário para a aplicação de acesso** alternar como Sim.
8. Clique em de **guardar** na parte inferior do ecrã.

Agora tem de atribuir utilizadores e/ou grupos à aplicação. Consulte [atribuir utilizadores a uma aplicação](active-directory-applications-guiding-developers-assigning-users.md) e [atribuir grupos para uma aplicação](active-directory-applications-guiding-developers-assigning-groups.md).
