# <a name="azure-machine-learning-studio-documentation"></a>Documentação do Azure Machine Learning Studio

O Azure Machine Learning Studio é uma ferramenta de colaboração, de arrastar e largar que pode utilizar para criar, testar e implementar soluções de análise preditiva nos seus dados.
Tutoriais, vídeos e modelos de exemplo mostram-lhe como criar e implementar modelos de aprendizagem automática com o Studio.
