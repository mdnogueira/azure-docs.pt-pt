# <a name="discover-and-share-tutorials-in-cortana-intelligence-gallery"></a>Detetar e partilhar tutoriais na galeria da Cortana Intelligence
[!INCLUDE [machine-learning-gallery-item-selector](../../../includes/machine-learning-gallery-item-selector.md)]

## <a name="tutorials"></a>Tutoriais
Galeria da Cortana Intelligence oferece várias [tutoriais](https://gallery.cortanaintelligence.com/tutorials) que pode ajudá-lo conceitos e as tecnologias do machine learning. Tutoriais também poderão descrevem métodos avançados para resolver problemas do machine learning.

## <a name="discover"></a>Descobrir
Para procurar tutoriais [na galeria](http://gallery.cortanaintelligence.com), em **mais**, selecione **tutoriais**.

![Selecione os tutoriais na home page da Galeria](./media/gallery-tutorials/select-tutorials-in-gallery.png)

O  **[tutoriais](https://gallery.cortanaintelligence.com/tutorials)**  página apresenta uma lista de tutoriais recentemente adicionados e populares. Para ver todos os tutoriais, selecione o **ver todos os** botão. Para procurar um tutorial específico, selecione **ver todos os**e os critérios de filtro, em seguida, selecione. Também pode introduzir termos de pesquisa no **pesquisa** caixa na parte superior da página da galeria.

Pode obter mais informações sobre um tutorial na página de detalhes do tutorial. Para abrir uma página de detalhes do tutorial, selecione o tutorial. No **comentários** secção numa página de detalhes do tutorial, pode comente, fornecer comentários ou fazer perguntas sobre o tutorial. Mesmo podem partilhar o tutorial com amigos ou colegas no Twitter ou LinkedIn. Também pode enviar por e-mail uma ligação para a página de detalhes do tutorial, para convidar outros utilizadores para ver a página.

![Partilhar este item com amigos](./media/gallery-how-to-use-contribute-publish/share-links.png)

![Adicionar os seus comentários](./media/gallery-how-to-use-contribute-publish/comments.png)

## <a name="contribute"></a>Contribuir
Pode adicionar um tutorial para a Galeria para ajudar a outros utilizadores a resolver um problema ou mais um conceito.

### <a name="create-a-tutorial"></a>Criar um tutorial

1. Inicie sessão para a galeria com a conta Microsoft.

2. Selecione a imagem na parte superior da página e, em seguida, selecione o nome.
  
    ![Selecione o nome de conta](./media/gallery-tutorials/click-account-name.png)

3. Selecione **Novo Item**.
  
    ![Selecione o "novo Item de"](./media/gallery-collections/click-new-item.png)

4. No **Descrição** página, para **tipo de ITEM**, selecione **Tutorial**. Introduza um nome para o tutorial, um breve resumo, uma descrição detalhada e as etiquetas que podem ajudar a outros utilizadores localizar o tutorial.
  
    ![Introduza as informações para o tutorial novo](./media/gallery-tutorials/create-tutorial-page-1.png)
5. No **seleção da imagem** página, selecione uma imagem que será apresentada com a coleção. Pode carregar o seu próprio ficheiro de imagem ou selecionar uma imagem as cotações. Escolha uma imagem que pode ajudar os utilizadores a identificar o conteúdo e o objetivo do tutorial.
  
    ![Introduza as informações para o tutorial novo](./media/gallery-tutorials/create-tutorial-page-2.png)

6. No **definições** página, para **visibilidade**, selecione se a coleção é **pública** (qualquer pessoa pode ver o tutorial) ou **Unlisted** (apenas as pessoas com uma ligação direta para o tutorial, podem visualizá-lo).
  
    > [!IMPORTANT]
    > Depois de definir um tutorial para **pública** e, em seguida, selecione **criar**, mais tarde não é possível alterar o tutorial para **Unlisted**.
    > 
    > 
  
    ![Selecione "Público" ou "Não listados"](./media/gallery-tutorials/create-tutorial-page-3.png)

7. Selecione **Criar**.

O tutorial está agora na galeria da Cortana Intelligence. Os tutoriais listados na sua página de conta, o **itens** separador.
